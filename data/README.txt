Bonjour et bienvenu dans l'Aide !

Utilisation :
	- Sélectionner un fichier .txt grâce au bouton "Fichier".
	- Apparition du nom du fichier sélectionné.
	- Analyser et Afficher ce fichier grâce au bouton "Analyser et Afficher".

Informations :
	Des informations concernant vos actions seront affichées dans la partie "Informations".

Affichage :
	Permet l'affichage du graph une fois analysé.

Options :
	Bouton Trajectoire :
		Permet de générer un fichier .txt du résultat des trajectoires trouvés.
	Bouton Screenshot :
		Permet de générer une image png (par défaut) du graph résultat.
	Bouton Effacer :
		Permet d'effacer l'affichage du graph.
	Bouton Aide :
		Permet d'ouvre une fenêtre d'aide. Ce contenu est disponible également
		dans un fichier README.txt.
